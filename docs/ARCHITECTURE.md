# Architecture map (code → responsibility)

A concise map of where each part of the mean field game (MFG) pipeline lives.

## Pipeline
- **Graph build**: `src/utils/graph_loader.py::load_macau_graph` (OpenStreetMap
  fetch/cache, projection, intersection consolidation) →
  `src/run_exp02.py::_build_subgraph` (13-node complete `DiGraph` via all-pairs
  shortest paths) → `src/run_exp05.py::_load_and_build_graph` +
  `_snap_to_graph_haversine`. Node registry (10 heritage attractions + 3 transit
  sources): `src/utils/attractions.py`.
- **HJB backward recursion**: `src/models/mfg_solver.py::MFGSolver.solve_hjb_backward`.
- **Fokker–Planck forward recursion**: `MFGSolver.solve_fp_forward`.
- **Damped fixed-point loop** `rho <- (1-lambda) rho + lambda * FP(HJB(rho))`:
  `MFGSolver.fixed_point_iteration` (`damping=lambda`).
- **Two-type compliance solve** (fraction `phi` follow the routing recommendation):
  `MFGSolver.fixed_point_iteration_compliance`.

## Parameters and what is data-driven vs assumed
- Learnable `{alpha_v}, beta, gamma` (log-space): `src/calibration/estimator.py::MFGParameters`.
- The softmax temperature `epsilon` is **fixed** (`solver.epsilon`, default 0.1), not fit.
- The arrival-rate `g(t)` **magnitude** comes from data
  (`run_exp05._daily_source_counts`); its **within-day shape is an assumed profile**
  (`src/utils/arrival_profiles.py`, default `gaussian`) — monthly data carries no
  intra-day information. See `docs/08_validity_scope.md`.
- **Only the monthly spatial attraction distribution is calibrated/validated**
  (held-out MAE 0.018, `run_exp05._evaluate_months`). All intra-day quantities
  (peaks, intervention magnitudes) inherit the assumed `g(t)` profile.

## Calibration
- Fit loop (one-step consistency gradient: no-grad fixed point → one HJB+FP step
  with autograd): inline in `src/run_exp05.py::run`; reusable class in
  `estimator.py::CalibrationEstimator`.
- Gradient correctness analysis (one-step vs unrolled vs implicit-function-theorem):
  `src/calibration/gradient_check.py` (see `docs/12_gradient_analysis.md`).

## Interventions
- Entrance metering on `g(t)`: `src/run_exp07.py::_meter_arrivals`.
- Routing bonus `eta`: `src/optimization/interventions.py::RoutingOptimizer` +
  `MFGSolver.routing_bonus`.
- Competing baselines (gravity, multinomial logit): `src/models/baselines.py`.
